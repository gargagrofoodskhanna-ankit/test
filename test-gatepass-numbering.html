Test Gate Pass Number Generation

Expected Format

Format: AAF/25-26/XXXX

Where:

  • AAF = Company/Organization prefix
  • 25-26 = Financial year (2025-2026)
  • XXXX = Auto-incrementing number starting from 0001

Examples:

  • AAF/25-26/0001
  • AAF/25-26/0002
  • AAF/25-26/0003
